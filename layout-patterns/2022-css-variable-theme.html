CSS 自定义属性（CSS Variables）

CSS变量（CSS Custom Properties）是CSS3引入的一项强大功能，它允许我们定义可重用的CSS属性值。通过在根元素上定义变量，我们可以在整个文档中重复使用这些值，并且可以通过JavaScript动态修改，从而轻松实现主题切换等功能。

使用CSS变量的好处：代码更易维护、主题切换更简单、减少重复代码、与JavaScript交互更方便。

💡 使用方式

定义变量：--primary-color: #667eea;

使用变量：color: var(--primary-color);

切换主题：document.documentElement.dataset.theme = 'dark'
