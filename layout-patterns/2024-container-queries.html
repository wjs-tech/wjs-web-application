Media Query vs Container Query

对比项 Media Query Container Query
查询对象 视口（viewport） 父容器（container）
适用级别 页面级响应式 组件级响应式
灵活性 全局统一，适合页面布局 局部自适应，组件真正复用
语法 @media (min-width: 768px) @container (min-width: 400px)
浏览器支持 全支持 Chrome 105+ / Safari 16+ / Firefox 110+
